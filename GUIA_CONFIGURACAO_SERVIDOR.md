# Guia de Configuração do Servidor GDIS
Este documento instrui como configurar a máquina que atuará como servidor central.
## Configuração de Rede
1.  **IP Estático**: Recomenda-se IP fixo.
2.  **Configuração do Atalho**: Edite `Atalho_Plataforma_GDIS_TEMPLATE.url` com o IP real.
## Monitoramento
Os logs ficam em `./data/atividades.log`.
